# **App Name**: LinkUp Landing Page

## Core Features:

- Header with Logo and Tagline: Displays the LinkUp logo and a concise tagline (e.g., 'Plan together, decide smart â€“ with LinkUp.') at the top of the page.
- App Overview: Provides a brief explanation of LinkUp's concept and value proposition in 1-2 paragraphs.
- App Screenshots: Showcases 3-4 mockup images of the app interface, highlighting key features like swiping and star rating.
- Feature Highlights: Lists key features with accompanying icons: Swipe for Choices, Star Voting, Collaborative Decisions, and Effortless Group Planning.
- Benefits Section: Presents the key benefits of using LinkUp through callouts or cards: Reduced Arguments, Perfect for Groups, and Smart Planning.
- Prerelease Registration Form: Collects user names and email addresses for prerelease notifications, storing the data in Firestore under a 'preregistrations' collection.
- Form submission feedback: Give the user positive feedback after form submission, using a tool to incorporate the entered name into the confirmation text, when appropriate.

## Style Guidelines:

- Primary color: Light sky blue (#87CEEB), reminiscent of clear skies and carefree vacations.
- Background color: Very light blue (#F0F8FF), maintaining a fresh and airy feel.
- Accent color: Soft green (#98FB98), for a touch of vibrancy and positive affirmation.
- Font pairing: 'Poppins' (sans-serif) for headlines and short text chunks; 'PT Sans' (sans-serif) for longer body text.
- Use clean, modern icons to represent key features. Consider a simple line-art style.
- Maintain a minimalist and responsive design with a mobile-first approach.
- Incorporate subtle animations or transitions on button hover and content loading for a modern feel.
